#### 복잡도
프로그램의 실행 속도는 프로그램이 동작하는 하드웨어나 컴파일러 등의 조건에 따라 달라짐.   
복잡도(complexity)란? 알고리즘의 성능을 객관적으로 평가하는 기준   
복잡도의 두가지 요소 
```
1. 시간 복잡도 (time complexity) : 실행에 필요한 시간 평가
2. 공간 복잡도 (space complexity) : 기억 영역과 파일 공간이 얼마나 필요한가 평가
```
